# Github 유저 찾기!

리액트를 활용해 만들었고, 깃허브 API를 사용해 깃허브의 유저 목록을 검색할 수 있는 사이트입니다!

---

## 🛒 주요기능

- 아이디 검색을 통해 유저를 찾을 수 있습니다.
- 유저를 클릭하면 상세보기를 볼 수 있고, 깃허브 페이지로 이동할 수 있습니다.
- 반응형 UI: 다양한 화면 크기에 대응할 수 있는 UI 설계.
- 클리어 버튼: 검색결과를 초기화하는 버튼이 있습니다.
